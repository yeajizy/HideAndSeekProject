## Description
전통주 판매 및 전통주 구독 서비스 쇼핑몰<br/>
한국에서 제조하는 여러가지 술들을 종합적으로 모아서 판매하는 쇼핑몰을 만들어 보고 싶었고, 요즘 트렌드에 맞게 구독 서비스도 포함시켜 단순히 구매하기만 할 수 있는 쇼핑몰이 아닌 정기적으로 구매를 하는 고객층을 확보하는 홈페이지를 제작하고 싶은 동기를 가지게 됨.
